## Project

**ARTIMES**
**A**utonomous **R**econnaissance and **T**hreat **I**ntelligence **M**ission **E**xecution **S**ystem

Platform:

* ROS 2 Humble
* Ubuntu 22.04
* Gazebo Classic
* Python (`ament_python`) package structure

# Phase 1 Goal

Create a ROS 2 workspace and a drone description package containing a minimal URDF model of the drone, then successfully display the model inside Gazebo.

**Success Criteria**

* ROS 2 workspace created
* Python package created
* Drone URDF created
* Workspace built successfully
* Drone spawned and visible in Gazebo

# ROS 2 Commands Used

Create workspace

```bash
mkdir -p ~/artimes/src
```

Create package

```bash
ros2 pkg create artimes_description \
--build-type ament_python \
--license Apache-2.0
```

Build

```bash
cd ~/artimes
colcon build --packages-select artimes_description
```

Source workspace

```bash
source install/setup.bash
```

Validate URDF

```bash
check_urdf urdf/artimes.urdf
```

Launch Gazebo

```bash
ros2 launch artimes_description display.launch.py
```
